# Portfolio screenshots (3D carousel)

Each project folder should contain PNG screenshots named **`1.png`**, **`2.png`**, etc.

The carousel uses **`1.png`** as the cover image for each project.

Example:

```
public/portfolios/foodpanda.hk/1.png
public/portfolios/crafterblue.com/1.png
```

Optional alternates (tried if `1.png` is missing): `2.png` … `5.png`, then `screenshot.jpg`.

Fallback placeholders: `public/portfolio/01.svg`–`06.svg`.
